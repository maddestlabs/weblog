# Welcome to TStorie Blog

Welcome to the TStorie Blog Engine! This is a terminal user interface (TUI) blog system built on top of the powerful TStorie framework.

## What is TStorie?

TStorie is a versatile engine for creating interactive terminal applications. It combines:

- **Layer-based rendering** - Organize your UI into composable layers
- **Markdown support** - Write content in familiar markdown format
- **Code execution** - Embed executable Nim code directly in your articles
- **Cross-platform** - Works on Linux, macOS, Windows, and even in the browser via WebAssembly

## Features of This Blog Engine

This blog engine demonstrates several key capabilities:

1. **Article Management** - Organize posts by date and category
2. **Navigation** - Browse articles with keyboard shortcuts
3. **Scrolling** - Read long articles with smooth scrolling
4. **Metadata** - Front matter support for titles, tags, authors, and more
5. **Fast Rendering** - Efficient TUI rendering at 60 FPS

## Getting Started

Use the following keyboard shortcuts to navigate:

- `↑/↓` - Scroll through article content or navigate article list
- `L` - Return to article list
- `ENTER` - Open selected article (when in list view)
- `Q` or `ESC` - Quit the application
- `PAGE UP/DOWN` - Scroll faster through articles
